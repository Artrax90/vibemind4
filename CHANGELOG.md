### Технические детали

- **Синхронизация:** каждые 2 минуты + при закрытии + ручная кнопка
- **Хранение паролей:** AES-GCM (PBKDF2 + salt)
- **Напоминания:** локальная SQLite + синхронизация с сервером
- **Шаринг:** ссылка генерируется через `crypto.randomUUID()`, сохраняется на сервере
- **Доски:** Canvas 2D рендеринг, фигуры, стрелки, фреймы

### Известные ограничения

- EXE ~149MB (превышает лимит GitHub 25MB)
- Vite кэш требует очистки перед каждой сборкой (`dist/`, `dist-electron/`, `node_modules/.vite`)
- Android: нет push-уведомлений (только локальные)
- Telegram-бот работает только на сервере
